# password-generator
Um gerador de palavras-passe robusto e seguro desenvolvido em Python, focado em conformidade com boas práticas de criptografia utilizando o módulo `secrets`.

# Características
- Utiliza o gerador de números pseudoaleatórios criptograficamente seguro (`secrets`).
- Garante a inclusão de pelo menos um caractere de cada conjunto selecionado (Maiúsculas, Números, Especiais).
- Interface CLI simples e direta.
